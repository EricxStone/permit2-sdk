# permit2-sdk

A permit2 SDK fork from Uniswap and get it on Ethers v6. Original SDK URL: https://github.com/Uniswap/sdks/tree/main/sdks/permit2-sdk

## Local Development

### Run Unit Tests
- `yarn test`

### Run Forge tests
- `yarn build`
- `yarn interop`
- `(cd permit2 && forge build)`
- `forge test`
